# Agentic IoT Platform

An IoT platform where edge devices don't just collect data — they reason, decide, and act autonomously using Claude AI agent skills, all remotely managed from a FlowFuse Dashboard.

## Concept

This project explores combining three technologies into a cohesive agentic IoT stack:

- **[FlowFuse Device Agent](https://github.com/FlowFuse/device-agent)** — edge runtime (Node-RED) on IoT hardware, remotely managed via snapshots
- **[FlowFuse Dashboard 2.0](https://dashboard.flowfuse.com/)** — operator control plane for monitoring and commanding AI agent fleets
- **[Claude AI (Anthropic)](https://docs.anthropic.com/)** — LLM reasoning layer that translates sensor data into structured device actions via tool use
